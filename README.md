# ejercicio_hackerRank
Planteamiento del ejercicio "Calcular la altura de un árbol" y su solución.

El ejercicio nos pide calcular la altura del árbol binario teniendo en cuenta que la altura se puede decucir por la cantidad de conexiones o nodos que tiene el árbol. Para poder solucionar este ejercicio se hace el uso de la recursión, donde va pasando por cada nodo desde la raíz hasta el ultimo de cada uno, así cuando se sabe la cantidad de nodos que tienen las ramas del árbol se comparan sus resultados para determinar cual tiene mayor altura, esto mismo se repites desde las ramas hasta la raíz teniendo de esta manera la altura del árbol.
